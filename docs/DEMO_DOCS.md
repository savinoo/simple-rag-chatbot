# Demo docs (local)

Create these files in the repo to demo citations + section-level refs:

## `docs/policies/returns.md`

```md
# Returns & Exchanges Policy

## Return Window
Opened items can be returned within **30 days** of delivery if they are in resaleable condition and include original packaging.

## Restocking Fee
Opened items may incur a **15% restocking fee**.

## Exceptions
Final sale items are not eligible for returns.
```

## `docs/sops/qc_checklist.md`

```md
# QC Checklist

## Pre-shipment
- Visual inspection
- Functional testing
- Photos (front/back/left/right)
```

Then update `manifest.example.yaml` to point at these local files.
